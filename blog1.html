HTML stands for HyperText Markup Language and it is a primary building block of creating a website.HTML is for creating Web pages.HTML elements are represented by tags. HTML tags identify content elements such as “heading”, “title”, “paragraph”, “table”, and so on. Browsers do not display HTML tags but use them to display the content of the page.

HTML Tags :

There are three main types of Tags in HTML; Opening Tag, Closing tag, and Content tag. Each tag has its different functions and methods. This all tags help us to run our program and documents without any error and correctly.

  1. HTML program is started using tag. This tag indicates that this is HTML code or text to be displayed in the browser.
  2. Second tag in any HTML document is tag. This tag represents the title of the webpage.
  3. The most important tag; the Body tag; Indicates all visible content on the web. This tag represents all sub-tags on the webpages.
  4. <p>- This tag represents the Paragraph on the webpage.
  5. <article> - This tag represents that, this is an Article on the webpage.
  6. <h1> - This tag represents the heading; <h1> represents the subheading and so go on.
  7. <b> - Defines the Bold text.
  8. <i> - Defines the italic text.
  9. <u> - Underline the text on the webpage.
  10. <table> - To create a table like excel on website.
